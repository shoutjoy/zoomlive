EZlive - P2P 화상 강의 및 판서 웹 애플리케이션

EZlive는 WebRTC와 PeerJS를 기반으로 하는 P2P(Peer-to-Peer) 화상 채팅 및 실시간 판서 애플리케이션입니다. 교사와 학생 간의 1:1 온라인 강의를 위해 설계되었으며, 별도의 서버 설치 없이 정적 호스팅만으로 동작합니다.

모든 기능은 단일 ezlive.html 파일에 포함되어 있습니다.

주요 기능

교사/학생 모드:

교사: 고유 ID와 비밀번호를 설정하여 강의실을 생성합니다.

학생: 전달받은 ID와 비밀번호로 강의실에 참여합니다.

고화질 화상/음성 통화: WebRTC를 통해 P2P로 연결되어 지연 시간이 짧은 1:1 통신을 지원합니다.

화면 공유: 교사 또는 학생이 자신의 전체 화면, 특정 창, 또는 브라우저 탭을 공유할 수 있습니다.

실시간 판서 도구 (화이트보드):

펜, 지우개, 포인터 기능

펜 색상 및 굵기 조절

실시간 동기화: 교사와 학생이 동시에 그리거나 볼 수 있습니다.

페이지 관리: 새 페이지 추가, 이전/다음 페이지 이동

저장: 현재 페이지(PNG, PDF) 또는 모든 페이지(PDF) 저장

파일 전송: 채팅을 통해 모든 종류의 파일을 P2P로 직접 전송할 수 있습니다.

실시간 채팅: 강의 중 텍스트 기반 소통이 가능합니다.

부가 기능:

채팅 기록 CSV 파일로 저장

내 화면/상대 화면 PIP (작게 보기)

상대 화면 전체화면 보기

오디오/비디오 켜기/끄기
